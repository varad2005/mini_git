Welcome to Mini GitHub! 🚀

            
This system helps you save and manage versions of your project. Here is how to use it:

            
            

                

                    
1. Start Tracking

                    
Click Init Repo to prepare your folder. This only needs to be done once.

                

                

                    
2. Prepare & Save

                    
Add files to stage them, then Commit to save a permanent snapshot. Every commit creates a unique ID.

                

                

                    
3. Teamwork (Merge)

                    
Use Branch to try new ideas safely. Use Merge to combine those ideas back into your main project. If two people changed the same line, we'll mark it for you to fix!

                

                

                    
4. Safety Net (Stash)

                    
Not ready to commit yet? Use Stash to "pause" your work. You can Pop it back later when you're ready.

                

                

                    
5. Visual History

                    
Switch to the Graph Visualization tab to see your project's family tree. It shows exactly how your branches grew and merged over time.
